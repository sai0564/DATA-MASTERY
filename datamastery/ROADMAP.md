# Roadmap

## Short Term
- **Level 2 (Data Cleaning)**: Handling NaN values, duplicates, and type casting.
- **Level 3 (Aggregation)**: GroupBys, window functions, and time-series analysis.
- **Achievements UI**: A global profile page for users to view earned DP and badges.

## Mid Term
- **Data Visualization**: Integrating Matplotlib/Seaborn support into Pyodide.
- **SQL Engine**: Support for SQL cells passing data directly to Pandas DataFrames.

## Long Term
- **Backend Sync**: Optional Supabase/Firebase synchronization for saving progress across devices (currently relies purely on robust `localStorage`).
